# Resume Website

This project is a simple resume website created using HTML, CSS, and JavaScript. It showcases personal information, education, work experience, skills, and contact details in a clean and organized manner.

## Project Structure

```
resume-website
├── src
│   ├── index.html        # Main HTML document for the resume website
│   ├── styles
│   │   └── style.css     # CSS styles for the resume website
│   └── scripts
│       └── script.js     # JavaScript for interactive features
├── README.md             # Documentation for the project
└── .gitignore            # Files and directories to be ignored by Git
```

## Getting Started

To run the resume website locally, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd resume-website
   ```

3. **Open the `index.html` file in your web browser**:
   You can simply double-click the `index.html` file or open it using your preferred web browser.

## Features

- Personal Information Section
- Education Section
- Work Experience Section
- Skills Section
- Contact Details Section

## Customization

Feel free to customize the content in `src/index.html`, adjust styles in `src/styles/style.css`, and add any interactive features in `src/scripts/script.js` to make the resume your own.
